Strata Mixer v1.1 beta — Windows installer project

Эта версия называется в окне просто: Strata Mixer.
Версия показывается аккуратным бейджем: v1.1 beta.

Что внутри:
- React/Electron приложение Strata Mixer
- Smart Render: авто/CPU/NVIDIA/Intel/AMD, безопасный откат на CPU
- режим нагрузки и параллельная обработка
- проверка ускорения и отображение реального кодировщика
- очередь файлов с сортировкой и удалением
- Donate, Справка, отчёты TXT/CSV, контроль веса и безопасные имена файлов
- сборка Windows installer через Inno Setup

Как собрать установщик:
1. Установи Node.js LTS.
2. Установи Inno Setup 6.
3. Распакуй папку в путь без кириллицы, например C:\StrataMixerBuild.
4. Запусти START_HERE_BUILD_WINDOWS_INSTALLER.bat.
5. Готовый установщик будет тут:
   installer-output\StrataMixer_Setup_1_1_beta.exe

Этот установщик уже можно отправлять пользователям. Им Node.js/Inno Setup/FFmpeg не нужны.
