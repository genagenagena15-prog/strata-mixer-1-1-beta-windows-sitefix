# Выпуск Strata Mixer v1.3.1 beta

В этой версии:
- Уже есть автообновление для Windows.
- В стартовом окне показывается текущая версия программы.
- Надпись в стартовом окне: "Проверка обновлений".
- Пустой предпросмотр показывает PNG-иконку на прозрачном фоне.
- Эффекты не применяются к placeholder-картинке, пока видео не загружено.

Как собрать:

1. Запусти:
   START_HERE_BUILD_WINDOWS_INSTALLER.bat

2. Готовый файл:
   installer-output\StrataMixer_Setup_1_3_1_beta.exe
