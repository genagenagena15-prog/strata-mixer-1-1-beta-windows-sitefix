# Выпуск Strata Mixer v1.2 beta

1. Собери установщик:
   START_HERE_BUILD_WINDOWS_INSTALLER.bat

2. Готовый файл:
   installer-output\StrataMixer_Setup_1_2_beta.exe

3. Создай GitHub Release:
   tag: v1.2.0-beta
   title: Strata Mixer v1.2 beta

4. Прикрепи файл:
   StrataMixer_Setup_1_2_beta.exe

5. После публикации release скопируй ссылку на .exe и обнови сайт/version.json:
   latest: 1.2.0-beta
   displayVersion: v1.2 beta
   windowsUrl: ссылка на GitHub Release .exe

6. Загрузи обновлённый сайт на Cloudflare Pages.
